# Jimmy Setup

Call the `jimmy_setup` MCP tool and follow the instructions in the response.

Write each file from the `commands` array to `.claude/commands/<filename>`. Overwrite if it already exists.

Report connection status, installed skills, and tell the user they can run `/jimmy` to see skills or `/jimmy [name]` to run one.
